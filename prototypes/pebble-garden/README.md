# Pebble Garden Mechanics Prototype

This is a static browser prototype for a simpler stone-based puzzle mechanic.

Open `index.html` in a browser.

Prototype goals:

- Test whether placing colored pebbles feels immediately understandable.
- Demonstrate group clearing with three or more connected pebbles.
- Show marked flower beds growing from seed bed to sprout to flower.
- Seed one opening sprout so the first clear blooms into a visible flower bed.
- Test a simple round target: bloom 4 marked beds before 14 moves run out.
- Show a calmer, more tactile direction than the route-based prototype.
- Keep the rules simple enough for a first iPhone game.

This is not the final visual design or production code.
