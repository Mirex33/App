# Signal Garden Mechanics Prototype

This is a static browser prototype for testing the core Signal Garden mechanic before building the native iOS version.

Open `index.html` in a browser.

Prototype goals:

- Show how path tiles are selected, rotated, and placed.
- Demonstrate source-to-source route completion.
- Demonstrate board clearing, scoring, and retry pacing.
- Keep the implementation simple enough to change quickly.

This is not the final visual design or production code.
